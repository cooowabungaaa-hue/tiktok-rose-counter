# Migration Guide: Personal PC Setup

職場PCで作成した「TikTok Rose Counter」を自宅の個人PCで起動するための手順です。

## 構成内容
このプロジェクトは **ポータブル版 Node.js** を含んでいるため、自宅のPCに Node.js をインストールする必要はありません。

## 起動方法（2つのパターン）

ご自宅のPC環境に合わせて、以下のいずれかの方法で起動できます。

### パターンA：そのまま使う（インストール不要）
もっとも簡単な方法です。私が用意した「ポータブル版Node.js」をそのまま使用します。
- フォルダ内の **`start.bat`** をダブルクリックするだけで起動します。
- 自宅PCにNode.jsをインストールする必要はありません。

### 友人に配布する場合（推奨：全自動パターン）
友人に渡す際は、以下の手順がもっとも親切です：
1. ご自身のPCで一度 **`start.bat`** を実行し、黒い画面に「起動しています」と出るまで待ちます（これで `node_modules` が作成されます）。
2. アプリが動くことを確認したら一度閉じます。
3. その **フォルダごとZIP圧縮** して友人に渡してください。
4. 友人はZIPを解凍して **`start.bat`** をダブルクリックするだけで、コマンド入力なしですぐに起動できます。

---

## 自宅で Antigravity に続きを頼む場合
もし自宅でもこの AI（Antigravity）を使用して開発を続けたい場合は、以下の手順を行ってください：

1. 自宅PCの Antigravity で新しいチャットを開始します。
2. コピーしたフォルダをワークスペースとして設定するか、「このフォルダにあるコードを読み込んで」と伝えてください。
3. `server.js` や `index.html` を読み込ませれば、そこから機能追加などが可能です。
