🌹 TikTok Rose Counter 🌹

かんたんセットアップガイド

1
セットアップ方法を選ぶ (どちらか一方でOK)

🌹 パターンA：しっかり使いたい方 (推奨)
PCに Node.js をインストールします。動作が安定し、起動も速くなります。

Node.js をダウンロード
💡 インストール後、フォルダ内の setup.bat を実行してください。必要な準備がすべて自動で行われます。

🌹 パターンB：インストールができない・制限がある方
職場のPCなど、インストールの許可（管理者権限）がない場合でも利用可能です。

💡 何もしなくてOK！ 次のステップでダウンロードする start.bat を直接実行してください。
2
ツールをダウンロードする

最新のツール一式（ZIPファイル）をダウンロードして、好きな場所に解凍（展開）してください。

📥 ツールをダウンロード (ZIP)
3
起動する

パターンAの方：setup.bat 完了後、start.bat で起動します。
パターンBの方：最初から start.bat をダブルクリックするだけでOKです。

🚀 起動すると自動的にブラウザが開きます。
※管理者権限の許可（ファイアウォール）画面が出ないよう対策済みですので、職場でも安心して実行できます。
4
使う！（待機モード対応）

起動すると自動的にブラウザが開きます。

  • CONNECT ボタンを押すと「待機モード」になります。
  • この状態で配信を始めると、自動的に接続されます！
  • 目標個数を入力して SET GOAL を押しておけば、接続時に自動で適用されます。
🍵 ツールを「待機」にしておけば、あとはスマホで配信開始ボタンを押すだけ。焦る必要はありません！
?
新しい機能が追加されたら？

作者から「アップデートしたよ！」と連絡があったら、フォルダの中にある update.bat をダブルクリックしてください。

自動的に最新のプログラムをダウンロードして、更新してくれます。Gitなどの難しい操作は不要です！

🔄 2回目からは、この「update.bat」を押すだけでOKです。
